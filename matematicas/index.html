Hoja de Fórmulas de Matemáticas del GED

Área

  • Cuadrado: A = s²
  • Rectángulo: A = l × w
  • Paralelogramo: A = b × h
  • Triángulo: A = ½ × b × h
  • Trapecio: A = ½ × h(b₁ + b₂)
  • Círculo: A = πr²

Perímetro y Circunferencia

  • Cuadrado: P = 4s
  • Rectángulo: P = 2l + 2w
  • Triángulo: P = s₁ + s₂ + s₃
  • Circunferencia: C = 2πr o C = πd

Superficie y Volumen

  • Prisma: SA = 2lw + 2lh + 2wh, V = lwh
  • Cilindro: SA = 2πrh + 2πr², V = πr²h
  • Pirámide: V = ⅓Bh
  • Cono: V = ⅓πr²h
  • Esfera: SA = 4πr², V = (4/3)πr³

Álgebra

  • Pendiente-intercepto: y = mx + b
  • Fórmula cuadrática: x = [-b ± √(b²-4ac)] / 2a
  • Pitágoras: a² + b² = c²
